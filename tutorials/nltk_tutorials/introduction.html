The provisoning of the VM installs NLTK, as well as two data sets (conll2000 and brown). This enables using some of the tutorials found in the NLTK Book.

At OSU, we have used the following chapters in class as walkthrough tutorials:

  • Chapter 5, Categorizing and Tagging Words
    • Class notes on the part-of-speech tagging tutorial.
  • Chapter 7, Extracting Information From Text
    • This requires megam to be installed from http://www.umiacs.umd.edu/~hal/megam/index.html. This is installed in the VM.
    • Helpful hint: If you want to print out the parse of a particular sentence in the test set, you need to extract the leaves from the CONLL tree, for example: 
      chunker.parse(test_sents[99].leaves()).draw()
